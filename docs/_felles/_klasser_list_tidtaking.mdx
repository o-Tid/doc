I dette feltet er alle klasser aktivt arrangement.

 ![Klasseliste-tidtaking](/img/klasseliste_tidtaking.png)

For hver klasse vises klassenavn, løypenavn, lengde og statuser for antall (Samme som i [topplinjen](#topplinje) )

Hver klasse kan klikkes på slik at listen i rute `1` viser bare aktuell klasse.

For å vise alle igjen kan man klikke på `Vis alle`
