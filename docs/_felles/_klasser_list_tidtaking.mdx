I dette feltet er alle klasser i arrangement.

 ![Klasseliste-tidtaking](/img/klasseliste_tidtaking.png)

For hver klasse vises klasse navn, løype navn, lengde og statuser for antall (Samme som i [topplinjen](#topplinje) )

Hver klasse kan klikkes på slik at listen i `1` viser bare aktuell klasse.

For å vise alle igjen kan man klikke på `Vis alle`
