Utstyr du trenger for å bruke O-Tid

- Tilgang til Internet (4G går helt greit). Kan fint bare dele internett med mobil eller om du har en mobilruter
:::tip[Tips]
 For at du skal være sikker på at det er bra nok (4g,5G) dekning der du skal ha samlingsplass (tidtaking skal skje) bør du sjekke dette på forhånd
:::
- Pc, Chromebook med nettleser (Chrome, Edge) og usb porter for klokke
:::tip[Tips]
 O-Tid er sjekket og brukt på.
 - Windows med Chrome og Edge
 - Chromebook med Chrome
 - Linux med Chrome (IKKE testet)
 - Mac med Safari (IKKE STØTTET) Safari støtter ikke WEB Serial

 Greit å sjekke at du har siste versjon av Chrome eller Edge
:::
- Emit tidtakings enhet (klokke). O-Tid støtter (eScan2,MTR4) _anbefalt eScan2_

:::tip[Tips]
 - Ved bruk av MTR4 så må MTR4 skrive ut strekktidslapp. (Kommer beskrivelse på hvordan det kan løses)
 - Skriver som følger med MTR4 støtter ikke utskrift av QR kode for link til Liveresultater
:::
