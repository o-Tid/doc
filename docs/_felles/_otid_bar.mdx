# Forklaring til topplinje


![Otidbar](/img/otid_bar_forklaring.png)


1. Påmeldt status. Antallet er de om ikke har kommet til sjekk/brikkesjekk (Ved klikk vises de i `1` for tidtaking)
2. Kommet status. Antallet som er sjekket og ute å løyper (Ved klikk vises de i `1` for tidtaking)
3. Mål status. Antallet som er kommet i mål eller avsluttet løpet sitt (Ved klikk vises de i `1` for tidtaking)
4. Totalt i arrangement (Ved klikk vises de i `1` for tidtaking)
5. Løpende klokke for PC, Chromebook
6. Løpsnavn og data
7. Klubb som er aktive
8. Kobling til server
9. Status for tilkoblet klokke (Ved klikk åpnes instillinger for klokke)
10. Status for printer for strekktidslapp (Ved klikk åpnes innstillinger for strekktidslapp printer)
11. Status for Liveresultater (Ved klikk åpnes innstillinger Liveresultater)
12. Status bruker (Ved klikk åpner innstillinger for bruker)
11. Hoved meny
